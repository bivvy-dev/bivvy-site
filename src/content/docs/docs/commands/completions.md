---
title: bivvy completions
description: Generate shell completion scripts
---

Generates shell completion scripts for tab-completion support.

## Usage

```bash
bivvy completions bash        # Bash completions
bivvy completions zsh         # Zsh completions
bivvy completions fish        # Fish completions
bivvy completions powershell  # PowerShell completions
bivvy completions elvish      # Elvish completions
```

## Supported Shells

| Shell | Value |
|-------|-------|
| Bash | `bash` |
| Zsh | `zsh` |
| Fish | `fish` |
| PowerShell | `powershell` |
| Elvish | `elvish` |
